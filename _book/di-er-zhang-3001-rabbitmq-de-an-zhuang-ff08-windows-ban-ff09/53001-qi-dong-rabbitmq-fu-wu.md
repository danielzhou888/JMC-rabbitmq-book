直接在命令行界面键入如下命令
```bash
net start RabbitMQ
```
![](/assets/1553601230093.png)

因为RabbitMQ默认启动的，当键入启动命令时，会出现如下情况，直接关闭RabbitMQ服务，在启动就行

![](/assets/1553601240037.png)


关闭RabbitMQ服务命令如下：
```bash
net stop RabbitMQ
```


![](/assets/1553601261740.png)
